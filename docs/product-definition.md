# 解決したい課題 / 潜在的なニーズ

ショートステイは、介護が必要な方が短期間だけ介護施設に宿泊し、必要な介護や日常生活の支援を受けるサービスです。利用者が日ごとに入れ替わり、衣類・日用品・薬・貴重品など多くの持ち物を持参するため、介護職員は入所時・退所時に持ち物確認を行う必要があります。

現場では、紙のチェック表や口頭確認で対応しているところも多く、持ち物の情報を正確に共有しづらいという課題があります。例えば「チェック柄の服」と記録しても、どの服を指しているのか、色やサイズなどの詳細までは十分に伝わりません。また、不足品が見つかった際には、家族への連絡や施設貸与品への切り替えといった対応が必要になりますが、忙しい業務の中ではその後の対応を把握しづらく、対応漏れや引き継ぎ漏れが起こりやすい状況があります。

このようなショートステイの持ち物確認業務では、持ち物の有無を確認するだけでなく、写真による実物の共有に加え、施設サービスとして利用できる日用品については、サービス利用有無と実際の持参状況のずれを把握し、必要な対応につなげられることが重要です。

# 対象ユーザー（誰の、どんな状況か）

- ショートステイ施設で入所時・退所時の持ち物確認を担当し、紙中心の業務に慣れている職員
- 日常業務が忙しく、複雑なデジタル操作に時間をかけたくない職員

# プロダクトのカテゴリー

- 持ち物管理 / 業務改善

# 提供する価値（ユーザーが得られる変化・体験）

- 持ち物確認を電子化することで、入所時・退所時の確認業務にかかる手間を減らし、職員が他の業務に充てられる時間を増やせます。
- 写真によって持ち物を具体的に記録できるため、職員間で確認対象を共有しやすくなり、退所時の取り違えや返却ミスを防ぎやすくなります。
- ショートステイは、毎週末の利用など、同じ利用者が繰り返し入所・退所することが多いため、前回の入力内容を初期表示できるようにすることで、毎回一から入力する手間を減らせます。
- 施設サービスとして利用できる日用品について、サービス利用有無と実際の持参状況の不一致を把握しやすくなることで、持参漏れを早く発見でき、家族への説明や職員間の引き継ぎがしやすくなります。

# 価値を実現するための最小限の機能（MVP）

- 事業所ごとの共通アカウントによるログイン
- 利用者情報の登録
- 利用者ごとの持ち物一覧の登録
- 前回の持ち物一覧を元にした初期表示
- 持ち物ごとの写真登録
- 入所時の持ち物チェック
- 退所時の持ち物チェック
- 日用品ごとのサービス利用有無の登録
- サービス未利用かつ未持参の場合のアラート表示
- 日用品不足時の状態管理（未対応、家族連絡済み、施設貸与、解決済み）
- 印刷用画面の表示

現場での利用を想定し、PC・タブレット・スマートフォンから利用できるWebアプリとして提供します。

# 既存の代替手段・競合との違い

- 紙のチェック表や口頭確認
  現場で導入しやすい一方で、持ち物の詳細を具体的に残しにくく、不足品が見つかった後の対応状況も別のメモや口頭で管理されやすいため、対応漏れや引き継ぎ漏れが起こりやすい課題があります。
- 既存の持ち物チェックアプリ
  既存サービスでは、入所時・退所時の持ち物チェックや写真記録、再利用などの基本機能が用意されています。([持ち物チェックアプリ](https://kaigosapuri.com/mochimono/))
- 本プロダクトでは、基本的な持ち物確認機能は踏襲しつつ、特に日用品に着目しています。一部の日用品は施設サービスとして利用できる一方、サービスを利用しない場合は本人持参が必要となります。そのため、サービス未利用であるにもかかわらず持参されていないケースをアラートで把握し、連絡・施設貸与・解決までの対応を追える点に特化しています。

# 技術的にこだわりたい・工夫したい点（無理ない範囲で）

- 紙中心の業務に慣れた職員でも使いやすいように、複雑な操作を避けたシンプルなUIを意識します。
- 保守しやすく、今後の機能追加にも対応しやすいような責務を分けた設計と分かりやすいデータ構造を意識します。
- 実務でも使われるようなDockerによる開発環境構築、GitHub ActionsによるCI、AWSを利用したデプロイなど、モダンな技術を無理のない範囲で取り入れます。

# なぜ自分がこれを作るのか（動機・原体験）

私の母は介護施設で働いています。職場は忙しく、人手不足の中で多くの業務を回しており、肉体的にも精神的にもつらそうな状況です。実際に母が「仕事が大変で辞めたい」と口にしていたこともあり、少しでも力になれることがないかと感じていました。

今の自分が作るプロダクトで現場の大きな問題をすべて解決できるとは思っていません。しかし、持ち物確認のように毎日のように発生する業務を少しでも楽にできれば、母の負担を減らせるのではないかと思い、このテーマを選びました。
